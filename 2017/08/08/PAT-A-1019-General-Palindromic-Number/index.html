1019. General Palindromic Number

原题链接

题目分析：

水题一道，题目写了一大片 + 很严谨，但其实说的就是回文判断再加上进制问题。获取十进制数字后，将其转为B进制后进行回文判断，简单的数字字符处理问题。

总结教训：

  • 一个是边界极限的问题：原文中一处提到输入的十进制数字N是 not negative，随后又提了一遍N的取值从0到10的九次方，这个点上卡了一个得分点。
  • 还有一个是方法使用的问题，一开始用字符串的时候，采用修正值+’0’的方法，但是这样的话遇到十进制以上就吃瘪，看了下别人的代码，换成vector后以后完美通过。

源代码：

#include <bits/stdc++.h>
using namespace std;
int main(void)
{
    int n, b;
    bool isPdm = true;
    vector<int> v;
    cin>>n>>b;
    if(n==0)
        v.push_back(0);
    while(n)
    {
        v.push_back(n%b);
        n/=b;
    }
    for(int i=0;i<v.size();i++)
    {
        if(v[i] != v[v.size()-1-i])
        {
                isPdm = false;
            break;
            }
    }
    if(isPdm)
        cout<<"Yes"<<endl;
    else
        cout<<"No"<<endl;
    for(int i=v.size()-1;i>=0;i--)
    {
        cout<<v[i];
        if(i)
            cout<<" ";
    }
    return 0;
}
