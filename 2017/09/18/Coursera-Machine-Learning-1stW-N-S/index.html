#Machine Learning

###定义：

远古版本

the field of study that gives computers the ability to learn without being explicitly programmed. (Arthur Samuel, 1959)

现代版本

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E. (Tom M. Mitchell, 1997)

举例：
玩跳棋的行为中
E 为游玩许多次跳棋获得的 experience
T 为游玩跳棋的 task
P 为程序赢得下场跳棋游戏的 probability

##Supervised Learning & Unsupervised Learning
任意机器学习问题可分为 Supervised Learning 或是 Unsupervised Learning

###两者区别

Supervised Learning

  • 给出数据集，已知正确的输出，并且在输入与输出存在某种关系
  • 分为 regression（输出值为连续的值，尝试构造连续函数） 及 classification（输出值为离散值，尝试将变量分类）

Unsupervised Learning

  • 给出数据，未知结果
  • 由数据变量中的内在关系将数据进行分类
